✈ OVERVIEW
🛬 What is ILS?

The Instrument Landing System (ILS) is a precision radio navigation system that helps aircraft land safely, especially during low visibility conditions such as fog, rain, or night.

It provides pilots with accurate guidance to align the aircraft with the runway and descend at the correct angle for landing.

ILS is one of the most important systems used in aviation to ensure safe and controlled landings.

⚙️ WORKING
❓ How Does ILS Work?

ILS works by transmitting radio signals from ground equipment to the aircraft. These signals guide the aircraft in two main directions:


1. Horizontal Guidance (Localizer)

  • Ensures the aircraft is aligned with the centerline of the runway
  • Located at the far end of the runway
  • If the aircraft deviates left or right, the system indicates correction

2. Vertical Guidance (Glide Slope)

  • Ensures the aircraft follows the correct descent angle (usually around 3°)
  • Located beside the runway
  • Helps the pilot descend smoothly towards touchdown

👉 The aircraft instruments display these signals, allowing pilots to adjust their path continuously.

🔧 COMPONENTS
📡 Main Components of ILS

📡 Ground Equipment

  • Localizer (LLZ) – Provides lateral (left-right) guidance
  • Glide Slope (GS) – Provides vertical (up-down) guidance
  • Marker Beacons (or DME) – Indicate distance from runway
    • Outer Marker
    • Middle Marker
    • Inner Marker

✈ Aircraft Equipment

  • ILS Receiver
  • Cockpit Display (CDI / HSI)
  • Autopilot (optional, for automatic landing)
💡 PRINCIPLE
🔬 Working Principle (Simple Explanation)
  • The ground station sends two overlapping radio signals.
  • These signals create a path in the air.
  • If the aircraft moves off this path:
    • Signal strength changes
    • Cockpit instruments show deviation
  • The pilot corrects the aircraft position accordingly.
👉 So basically:   ILS = Radio Signals + Aircraft Instruments = Safe Landing Path

PAPI Lights System

Precision Approach Path Indicator — runway ke paas 4 light units jo pilot ko sahi glide path pe guide karte hain.

💡 PAPI Live Simulation — Glide Path Indicator
White = Too High
Red = Too Low

💡 OVERVIEW
🔦 How Do PAPI Lights Work?

PAPI consists of four light units placed in a row beside the runway.

Each light can appear either:

⬜ White → Aircraft is too high

🔴 Red → Aircraft is too low

👉 Ideal condition: 2 White + 2 Red
✈ INDICATIONS
✈ Indications to Pilot
4 White → Too high
TOO HIGH
3 White, 1 Red → Slightly high
SLIGHTLY HIGH
2 White, 2 Red → ✅ Correct path
CORRECT ✅
1 White, 3 Red → Slightly low
SLIGHTLY LOW
4 Red → Too low (Danger)
